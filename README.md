# EE3111_Powerplay_Predictor
A course project using linear models for IPL score prediction

Statistical model to predict the score at 6 overs based on ball-by-ball data of first 3 overs

•	Developed a Multiple Linear Regression model to predict the total runs scored by the batting team at the end of the 6.0 over powerplay, using ball by ball sequence data from the first 3.0 overs using engineered features, batter bowler pair data, wickets fallen, extra runs and analysis of yearly trends.

•	Over the course of 2 rounds with 15 matches each of out of sample data, achieved RMSE of 12.05 and placed [9th](https://docs.google.com/spreadsheets/d/1lt6x7aMItXmXDegEmXYwJGmXYj8LfgIm/edit?gid=1852634179#gid=1852634179)

•	Main.py refers to the submission made that obtained the above results while the training file represents the analysis done on the data
